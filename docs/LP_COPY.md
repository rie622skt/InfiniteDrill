# LP 文言・コピー仕様

LP 各セクションで使用する文言を定義する。実装時はこのファイルを参照し、デザインと同期する。

---

## ヒーロー

- **メインコピー**: 二級建築士・構造力学を、スキマ時間で攻略。
- **サブコピー**: 梁・断面・トラス・ラーメンまで、1問から始められるドリル。
- **CTA ボタン**: 無料で始める

---

## 特徴（4ブロック）

1. **見出し**: 1問から、スキマ時間で  
   **本文**: 重い参考書は不要。通勤・休憩中に1問ずつ、自分のペースで続けられます。

2. **見出し**: 初級〜上級・総合で実力に合わせた出題  
   **本文**: 初級で基礎を固め、中級で応用。上級では張り出し梁・ラーメン・複雑な断面も。総合モードで実力判定できます。

3. **見出し**: 解説付きで理解が深まる  
   **本文**: 正解・誤答ごとに解説を表示。梁問題では曲げモーメント図・せん断力図も確認できます。

4. **見出し**: 診断モードで苦手を可視化  
   **本文**: 全カテゴリを解くとレーダーチャートで弱点が一目瞭然。苦手克服モードで集中的に演習できます。（※診断モードは有料コンテンツです）

---

## 出題カテゴリ

- **セクション見出し**: 幅広い単元をカバー
- **サブ**: 二級建築士の構造力学で出題される主要テーマを網羅しています。

| カテゴリ | 短い説明 |
|----------|----------|
| 梁 | 単純梁・片持ち・張り出し×集中・等分布 |
| 断面 | 断面係数・断面二次モーメント・図心 |
| 曲げ応力度 | σ = M/Z、複合応力度、短柱、せん断 τ |
| 座屈 | 座屈長さ・荷重比率 |
| トラス | ゼロメンバー・軸力計算 |
| ラーメン | 静定ラーメン（3ヒンジ門形） |
| たわみ | たわみ・たわみ角の比較 |

---

## 診断モード

- **見出し**: 診断モードで弱点を把握
- **本文**: 全13カテゴリを解くと、レーダーチャートで得意・苦手が可視化されます。診断結果に基づき苦手克服モードで集中的に演習可能です。
- **有料注記**: 診断モードは上級モードとセットで **500円（買い切り）** でご利用いただけます。一度の購入で永久に利用可能です。

---

## 料金

- **セクション見出し**: 料金
- **無料**: 初級・中級のみ（まずは無料で試せる）
- **500円（買い切り）**: 上級モード＋診断モードを一度の購入で永久利用。月額ではなく一度きりの支払い。追加料金なしでずっと使えます。

### 料金表

| 項目 | 無料 | 500円（買い切り） |
|------|------|-------------------|
| 初級・中級 | ○ | ○ |
| 上級・総合 | − | ○ |
| 診断モード | − | ○ |

- **CTA（料金ブロック内）**: 500円で上級・診断をアンロック（課金実装前は「近日対応」と表示）

## FAQ

1. **Q**: 二級建築士の試験範囲に合わせていますか？  
   **A**: はい。二級建築士の構造力学で出題される梁・断面・応力度・座屈・トラス・ラーメン・たわみをカバーしています。

2. **Q**: 500円払うと何が使えますか？  
   **A**: 上級モード（張り出し梁・ラーメン・中空断面など）と診断モード（全カテゴリの実力診断・レーダーチャート）が永久に利用可能になります。

3. **Q**: 月額ですか？買い切りですか？  
   **A**: 買い切りのみです。500円のお支払いは一度だけ。追加料金はかかりません。

4. **Q**: オフラインで使えますか？  
   **A**: アプリ内の出題・解説はオフラインでも利用可能です。課金処理時のみ通信が必要です。
